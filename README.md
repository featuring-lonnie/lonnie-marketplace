# Lonnie's Marketplace

Personal Claude Code plugin marketplace.

## Plugins

| Plugin | Description |
|--------|-------------|
| [workflow-manager](https://github.com/featuring-lonnie/workflow-manager-plugin) | Unified workflow management across Slack, Confluence, Jira, GitHub, and Google Calendar |

## Installation

```bash
# Add marketplace
/add-marketplace https://github.com/featuring-lonnie/lonnie-marketplace

# Install plugin
/install-plugin workflow-manager@lonnie-marketplace
```
